Git

  • git status: checks what branch we are currently working on
  • git checkout branch-name: switches to named branch
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -a: stages all edited files/revisions for commiting
  • git restore --staged'file-name': to unstage a file to commit
  • git commit -m 'comment/description': commits all changes added, effectively saves. Do not switch branches without commiting changes
